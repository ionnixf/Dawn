# Brand Assets

This directory contains Dawn brand assets.

## Logo

To generate the logo, use the prompt below with **DALL·E 3**, **Midjourney 6**, or **Ideogram 2.0**.

### Recommended model

| Model | Why |
|-------|-----|
| **Ideogram 2.0** | Best at rendering crisp, readable text in logos |
| **DALL·E 3** | Good for photorealistic brand boards, weaker at text |
| **Midjourney 6** | Excellent visual quality, weaker at precise text |

### Prompt

```
A premium brand logo for "Dawn" — a developer-focused browser start page with a warm terminal aesthetic.

Logo concept: A minimalist geometric lettermark "D" fused with a sunrise/horizon motif. The "D" is constructed from a partial circle (sun rising) and a vertical terminal-prompt-style line creating the stem. The mark sits above or beside a clean sans-serif wordmark "DAWN" in spaced uppercase.

The entire mark uses a warm terracotta (#C15F3C) on a very dark charcoal (#1A1815) circular or rounded-rect badge. Designed for dark-mode interfaces. Clean, scalable, modern — no gradients, no glows, no generic AI slop. Should look like it belongs on a developer tool or terminal product.

Style: minimal, geometric, symbolic, premium, Dark Developer / Builder mode.
Tagline underneath (optional): "start here" in a thin monospace type at a smaller size.
```

### Files needed

| File | Content |
|------|---------|
| `logo-dark.svg` | Logo on transparent bg, light mark for dark surfaces |
| `logo-light.svg` | Logo on transparent bg, dark mark for light surfaces |
| `logo-badge.svg` | Logo inside rounded square badge for favicon / app icon |
